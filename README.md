# 🎓 Academy Management System

A comprehensive **Console-Based Academy Management System** developed in **C++** using **Object-Oriented Programming (OOP)** principles. This project simulates the core operations of an educational institute by providing separate functionalities for **Admin**, **Teacher**, and **Student** roles.

The system demonstrates the practical implementation of OOP concepts such as **classes, objects, encapsulation, friendship, file handling, operator overloading, and modular programming** while maintaining persistent data using text files.

---

## 📌 Features

### 👨‍💼 Admin Module

* Add new students
* Remove existing students
* Add new teachers
* Remove teachers
* Manage student fee status
* Manage teacher salary status
* View academy records

### 👨‍🏫 Teacher Module

* Secure teacher login
* Assign and update student marks
* View salary payment status
* Manage academic records

### 👨‍🎓 Student Module

* Secure student login
* View examination results
* Check fee payment status
* Access subject-wise marks

## 🚀 Getting Started

### Prerequisites

* C++ Compiler (G++, MinGW, MSVC, etc.)
* Code Editor (Visual Studio Code, Code::Blocks, Visual Studio)

### Clone the Repository

```bash
git clone https://github.com/your-username/academy-management-system.git
```

### Compile

```bash
g++ test.cpp -o academy
```

### Run

```bash
./academy
```
